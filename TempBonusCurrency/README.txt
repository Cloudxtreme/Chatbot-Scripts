#####################
#   Redeem Script   #
#####################

Description: Let users spend points to increase the payouts for everyone in chat for a set duration
Made By: Castorr91
Website: https://www.twitch.tv/castorr.91

#####################
#       Usage       #
#####################
Castorr91: !bonus
Castorrbot: castorr91 paid 300 to give everyone in chat 15 bonus points every 30 seconds for the next 300 seconds
