#      Usage        #
#####################
This is using default values, if commandname is changed you use that instead.

!roulette <option> <amount>
Bet selected amount on chosen option

!roulette <option> all
Bet all your points on a chosen option

!roulette
Shows information on how to spin the roulette wheel
