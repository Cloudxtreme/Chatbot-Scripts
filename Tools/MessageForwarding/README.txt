#####################
#   Limitations     #
#####################

The bot must be connected properly to discord before the script will work

The script can only scan the Discord server the channel is connected to for messages to forward.

The script will forward messages to the discord chat the bot is connected to.

The bot cannot respond to its own messages so bot responses will NOT be forwarded.
